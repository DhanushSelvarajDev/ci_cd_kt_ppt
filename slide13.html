Trigger (Lines 3-7)

Defines when the pipeline runs. Here, it triggers on pushes to main or dev, and pull requests to main.

Environment (Line 11)

Specifies the OS environment. ubuntu-latest is a fresh GitHub-hosted runner for every job.

Steps & Actions (Line 13+)

Sequential tasks:
1. uses: Pre-built actions (checkout code, install Node).
2. run: Execute shell commands (npm install, test, build).
